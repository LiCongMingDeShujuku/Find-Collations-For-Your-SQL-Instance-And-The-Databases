## 中文
这是获取SQL Server排序规则（实例排序规则）和数据库排序规则的一些SQL逻辑（logic）。

## English
Here’s some sql logic to get the SQL Server Collation ( Instance Collation ), and the Database Collation.

---
## Logic
```SQL
use master;
set nocount on
 
declare @mydatabasename     varchar(255)
set     @mydatabasename     = 'msdb'  -- I'm using MSDB as an example
 
select
    'server name'           = serverproperty('servername')
,   'server collation'      = convert(varchar, serverproperty('collation'))
,   'database name'         = @mydatabasename
,   'database collation'    = databasepropertyex(@mydatabasename, 'collation');


```
